Introducción

¿Qué es el desarrollo web?

El desarrollo web es el proceso de crear y mantener sitios web para el Internet. Esto se lleva a cabo con tecnologías del lado del servidor y del lado cliente. Usando diversos lenguajes de programación, los desarrolladores web se especializan en crear la apariencia y el funcionamiento del sitio para darle una experiencia impecable al usuario.

Hay tres tipos de especialización de desarrollador web: desarrollador front-end, desarrollador back-end y desarrollador full-stack. Los desarrolladores de aplicaciones para el usuario son responsables del comportamiento y las imágenes que se ejecutan en el navegador del usuario, mientras que los desarrolladores de aplicaciones para el fondo se ocupan de los servidores.

“Los desarrolladores web son como duendes con poderes: nunca los ves, pero son los que hacen que todo esté bien y funcione de manera rápida y eficiente.” Fuente: OpenClassrooms

Lo que debes saber

  • Cliente - "El lado del cliente" significa que la acción tiene lugar en la computadora del usuario (el cliente). Un cliente solicita información del servidor.
  • Servidor - "El lado del servidor" significa que la acción tiene lugar en un servidor web. Un servidor llena la solicitud de los clientes.
  • Tecnologias - lenguajes de programación, lenguajes de marcado y hojas de estilo.
  • Lenguajes - HTML, CSS, DOM, JavaScript, PHP, Ruby, Python, Java, .NET
  • Una pagina web es realmente un documento de texto.

Front-end

"Front-end" se traduce directamente a el fin delantero o la parte interfaz, es la parte que ve el usuario. El desarollo front-end es la práctica de convertir datos a una interfaz gráfica, con el uso de HTML, CSS y JavaScript, para que los usuarios puedan ver e interactuar con esos datos.

HTML (Hyper Text Markup Language) es la columna vertebral de cualquier proceso de desarrollo de un sitio web, sin el cual no existe una página web. Es el código HTML que proporciona un marco general de cómo se verá el sitio. HTML fue desarrollado por Tim Berners-Lee en 1993. La última versión de HTML se llama HTML5 y fue publicada el 28 de octubre de 2014 por recomendación W3. Esta versión contiene formas nuevas y eficientes de manejar elementos como archivos de video y audio.

<!DOCTYPE html> <html> <head> <title>Titulo</title> </head> <body> <p>Hello World</p> </body> </html>

Las hojas de estilo en cascada (CSS) controlan el aspecto de presentación del sitio y permiten que su sitio tenga su propia apariencia única. El CSS le da forma, color, y posicion a una pagina.

p { font-size: 120%; color: gray; }

JavaScript (JS) es un lenguaje de programación imperativo basado en eventos (a diferencia del modelo de lenguaje declarativo de HTML) que se utiliza para transformar una página HTML estática en una interfaz dinámica. El código JavaScript puede usar el Modelo de objetos de documento (DOM), proporcionado por el estándar HTML, para manipular una página web en respuesta a eventos, como la entrada del usuario.

<script> alert( 'Hello, world!' ); </script>

Back-end

Tambien conocido como la capa de acceso a datos, "back-end" es es la parte o rama del desarrollo web encargada de que la lógica de una página funcione. Consiste en el conjunto de acciones que pasan dentro de una web, pero que no podemos ver. Por ejemplo, la comunicación con el servidor. Fuente: Nestrategia

Desde el "back-end" se llevan a cabo todas las funciones que hagan más simple el proceso de desarrollo. Las acciones de lógica y las conexiones con las bases de datos. También se mantiene la seguridad de los sitios web.Fuente: Nestrategia

Lenguajes más utilizados dentro del Back-End son:

  • PHP: Muy utilizado en algunos de los principales gestores de contenido como WordPress.
  • Ruby: También se utiliza su framework Ruby on rails.
  • Python: Se usa comúnmente con Django como framework.
  • Node.js: Más utilizado en los últimos tiempos, ya que utiliza un lenguaje similar al del lado del cliente (JavaScript).
    • Fuente: Nestrategia

Full-stack

El desarrollo web "full-stack" es la práctica de trabajar tanto en el "front-end" como en el "back-end" de un sitio web. Los desarrolladores tienen experiencia en la creación de interfaz de usuario y también tienen un gran conocimiento en un lenguaje de programación que se utiliza para manejar la lógica de la aplicación, el back-end.

Las ventajas de un desarrollador full-stack son la adaptabilidad y flexibilidad: pueden ponerse a trabajar fácilmente en tareas de front-end o back-end, buenas posibilidades de ser empleado, conocer la estructura completa de lógica y del diseño, y tener una mejor visión de un programa o aplicación.
